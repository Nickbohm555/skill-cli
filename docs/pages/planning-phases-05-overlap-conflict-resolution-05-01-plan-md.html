Return sorted findings and overall severity in OverlapReport; never auto-resolve conflicts in detection code. go test ./internal/overlap -run Detect -v passes with fixtures for no-overlap, medium overlap, high overlap, and exact-content-match cases. Given candidate + installed profiles, the system consistently emits explainable overlap findings and severity for OVLP-01.
